# Безопасность при деплое

- Пароли, которые когда-либо отправляли в чат или почту, считайте **скомпрометированными**. Сразу выполните на сервере `passwd` (для `root` и других учёток с доступом).
- Переведите вход на **SSH-ключи** (`ssh-keygen`, `ssh-copy-id` или вручную добавить `~/.ssh/authorized_keys`). По возможности отключите `PasswordAuthentication` в `sshd_config` после проверки ключей.
- Секреты и `.env` с реальными значениями **не коммитьте**. На сервере храните `/etc/porubly/admin-front.env` с правами `chmod 600` и владельцем `root`.
